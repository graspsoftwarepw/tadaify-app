Informational notification (no token) sent after a third-party identity is linked to an existing tadaify account via OAuth. Real-impl placeholder: {{ .Provider }} for the provider name (Google / Apple / etc.). This is a security trail email — the user already authorized the link in-flow; this email arrives as a record so an unauthorized link triggers an immediate review. Below: rendered in a light email client; plain-text fallback (multipart/alternative) at the bottom.

What plain-text-only clients see

Some clients (terminal mail readers, accessibility tools, SMS gateways, low-bandwidth previews) render only the text/plain part. The fallback contains the same notification in clear ASCII — no markup, no CSS, no images.
