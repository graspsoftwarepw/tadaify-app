These toggles control which cookies are set on visitors to your public page (creator-side control). Visitors can still opt out individually via the consent banner — disabling a category here removes it from the banner entirely.

Essential Always on
Required to render the page, log visitors in, run the cart, and prevent CSRF. Cannot be disabled — without these the page does not work.
  • tadaify_session · session, HttpOnly · 30 days
  • tadaify_csrf · session · cleared on tab close
Analytics Default ON
First-party page-view + click tracking that powers your Insights dashboard. No third-party trackers, no IDFA. Visitors can always opt out via the cookie banner.
  • td_pv · 1st-party · 365 days · page views, no fingerprinting
  • td_clk · 1st-party · 365 days · click counts per block
Marketing Default OFF
Third-party trackers if you embed Meta Pixel, Google Ads, TikTok Pixel, etc. Off by default per privacy-by-design (GDPR Art. 25). Turning this on requires you to update your Cookie Policy.
  • _fbp (Meta Pixel) · 90 days · only if you embed it
  • _gcl_au (Google Ads) · 90 days · only if you embed it
Personalization Default OFF
Stores visitor preferences across visits — last-viewed theme, A/B test bucket assignment, language preference. No identifying data.
  • td_pref · 1st-party · 365 days · theme + language
