1. Install

npm install -g @tadaify/mcp

2. Add to Claude Desktop config

// ~/.config/claude/claude_desktop_config.json
{
  "mcpServers": {
    "tadaify": {
      "command": "tadaify-mcp",
      "args": [],
      "env": {
        "TADAIFY_API_KEY": "sk_live_•••abc123"
      }
    }
  }
}

Change handle?

Your tadaify slug changes from tadaify.com/alexandra to tadaify.com/—.

What happens to the old URL
For 30 days, tadaify.com/alexandra will 301-redirect to your primary page: → https://alexandra.com/ (your active custom domain) After 30 days the old slug is released and someone else can claim it.
Custom domains stay bound to your account — they're not tied to your handle, so alexandra.com keeps working with no DNS change. Only the tadaify.com slug needs a redirect.

You can change your handle again in 30 days.
