Custom CSS BUSINESS

For when the controls above don't go far enough. Wrapped in a scoped layer so you can't break the editor or sign-in flows.

theme.custom.css
scoped @layer custom 
/* Your CSS lands inside @layer theme-custom { ... } */
/* The 4 palette swatches are exposed as CSS custom properties: */
:root {
  /* readable in your overrides */
}

.creator-page .block {
  border-radius: 22px;
  box-shadow: 0 8px 24px rgba(17,24,39,0.06);
  transition: transform 160ms ease;
}
.creator-page .block:hover {
  transform: translateY(-2px);
}

.creator-page .pf-name {
  background: linear-gradient(90deg, var(--theme-primary), var(--theme-accent));
  -webkit-background-clip: text;
  color: transparent;
}
