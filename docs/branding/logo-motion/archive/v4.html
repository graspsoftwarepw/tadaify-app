What changed vs v3

Three deliberate changes from v3 → v4

  • Wave is now rocking side-to-side, not overshoot-pop — v3's warm orb arrived at the edge point with a spring overshoot (travel past the edge then snap back), which read as a "bounce" or "pop" rather than a greeting gesture. v4 replaces the overshoot with a dedicated wave beat: 3 cycles of perpendicular oscillation (±10 SVG units) while the orb stays at the edge. The wave motion is entirely separate from the travel motion.
  • Slower, more deliberate tempo (~11.6s vs ~6.4s) — adding the wave beat at each of the 4 edge points adds ~1.48s per greeting. The creator is now "present" at each edge for nearly a second, which matches the "greeting the audience" metaphor — not urgent, not a loading spinner, but a warm acknowledgement.
  • Metaphor cemented: creator on stage greeting the audience — yellow circle = the creator (the person who owns the page), purple circle = the stage. The creator walks to each edge, waves hello to the audience on that side, returns to center, repeats. The long rest at center is the "brand mark" state — both presence and readiness.

Accessibility

The warm orb animation is suppressed via prefers-reduced-motion: reduce. When detected, all Web Animations API instances are not started and the warm orb remains statically centered on the large orb — the fully legible brand mark with no movement. Favicon sizes (16px, 32px) always render static regardless of preference, as the wave motion is illegible below ~40px.
