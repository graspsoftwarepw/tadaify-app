# Custom domain included in Creator tier (1 domain per DEC-038)

## Context

Custom domain is the primary revenue lever and conversion signal for tadaify. Creator
tier ($7.99/mo after DEC-279) includes 1 custom domain as part of the plan — this is
the core justification for the $6 premium above the $1.99 domain add-on price alone.

## Decision

1 custom domain is included in the Creator tier plan price. No separate add-on charge
for the first domain on Creator or Pro tier. Business includes 10 custom domains (agency use).

Universal domain add-on: any user on any tier (including Free) can add extra custom
domains at $1.99/mo each via F-DOMAIN-ADDON-001.

## Consequences

- Creator positioning: "Your own domain included" is the lead benefit.
- Free creators can still get a custom domain via the $1.99/mo universal add-on.
- Pro includes 1 domain (same as Creator); Pro differentiator is the 8 power features.
