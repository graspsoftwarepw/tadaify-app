## B. Feature unlocks by tier (most important only)

### Linktree
- **Free → Starter ($8):** email collection block, custom backgrounds, link scheduling
- **Starter → Pro ($15):** advanced analytics (geo, device, referrer), Mailchimp/Zapier integrations, branded short URL, Pro animations, Meta/TikTok Pixel
- **Pro → Premium ($35):** custom domain, custom CSS, advanced commerce (0% platform fee), priority support, SEO meta, API access (beta dev program)
- **Enterprise (custom):** SSO (not confirmed), audit log, white-label, multi-brand Workspaces (beta)

### Beacons
- **Free → LiB Pro ($10):** custom domain, remove branding
- **Free → Store Pro ($30):** 0% fee on products, native cart, course hosting
- **Free → Email Pro ($13):** subscriber import, automation
- **All-in-One / Business ($90):** all above + AI Brand Outreach, daily media kit updates

### Stan.Store
- **Creator ($25):** store, unlimited products, courses, bookings, community, AutoDM, subscriptions
- **Creator Pro ($99):** upsells, order bumps, email marketing broadcasts, discount codes, affiliate program, payment plans, ad pixel tracking

### Carrd
- **Free → Pro Lite ($9/yr):** custom domain, forms, Google Analytics widget
- **Pro Lite → Pro Standard ($19/yr):** more widgets (Stripe, PayPal, Gumroad, Mailchimp, Calendly, Typeform), Google Analytics premium
- **Pro Standard → Pro Plus ($49/yr):** 25 sites, password protection, custom domain fonts, canonical URLs, downloadable site files, update frequency

### Later Linkin.bio
- **Starter ($16.67):** basic link-in-bio, content scheduling
- **Growth ($30):** shoppable grid, multi-link-per-post (up to 5), team seats, advanced analytics
- **Advanced ($53.33):** API access, Shopify product sync, full workspace, multi-brand
- **Agency (custom):** SSO, SOC2 DPA, white-label

### Taplink
- **Free:** basic page, limited blocks
- **Pro ($3):** 20 blocks, messenger buttons, themes
- **Business ($6):** Forms, Payments, Booking, Pixel
- **Premium ($9):** everything + custom CSS-like styling + custom domain

### Milkshake
- **Free:** full features, Milkshake branding, app-only
- **Lite ($2.99):** remove branding
- **Pro ($6.99):** advanced analytics, multiple sites, 1 custom domain
- **Pro+ ($59.99/yr):** multi-site + power features

### Campsite.bio
- **Free:** unlimited links, templates, basic analytics
- **Pro ($7):** custom domain, advanced analytics, embed, Mailchimp, scheduling, Pixel
- **Pro Plus ($24):** multiple profiles, team seats, API + webhooks, organisation features
- **Organization (custom):** white-label, SSO (not confirmed)

### Bio.link
- **Free:** no branding, unlimited links, basic analytics
- **Premium ($5):** custom domain, advanced analytics, more themes

### Lnk.Bio
- **Free:** unlimited links, 368+ themes, basic analytics
- **Mini ($0.99/mo OR $9.99 OTP):** remove branding
- **Unique ($24.99 OTP):** all features, custom domain, 44k Google Fonts, lifetime
- **Agency:** bulk profile management

## D. Branding-removal paywall comparison

This is the #1 complaint in user reviews (per `competitors.md`). Where does branding first come off?

| Tier | Cost/mo | Brand removal available? |
|------|---------|--------------------------|
| Free | $0 | Bio.link ✅ / others ❌ |
| Under $3 | $0.99–$2.99 | Lnk.Bio Mini, Milkshake Lite, Taplink Pro ($3) ✅ |
| $5–$8 | Bio.link Premium, Linktree Starter | Bio.link ✅ / Linktree ❌ (still branded on Starter for some elements) |
| $10+ | Beacons Pro, Linktree Pro+ | All ✅ |

**Implication for tadaify:** Bio.link and Milkshake Lite have already eaten the "remove-branding-cheap" angle. The "never-branded" positioning is stronger than "branding-free-on-paid", but Bio.link is the direct competitor.

## E. Transaction-fee structure

| Competitor | Free-tier commerce fee | Paid-tier commerce fee | Who pays Stripe? |
|------------|------------------------|----------------------|-------------------|
| Linktree | 12% | 0% on Premium ($35) | User |
| Beacons | 9% | 0% on Store Pro ($30+) | User |
| Stan.Store | N/A (no free) | 0% | User |
| Taplink | 0% | 0% | User |
| Bio.link | 0% | 0% | User |

**Takeaway:** Linktree's 12% is the outlier. Taplink + Bio.link already charge 0% at every tier. For tadaify, 0% is the default industry position — not a differentiator unless paired with *native checkout on free*, which Taplink already does.

## F. Annual-vs-monthly discount

Roughly 15-25% discount for annual billing across the board. Stan.Store is the cleanest example: $29 monthly → $25/mo when billed annually = $48 annual savings. Linktree, Beacons, Campsite all use similar ~20% annual discounts. Carrd, Taplink already advertise per-year.

## G. Price-lock history

- **Linktree raised prices Nov 2025** (Pro from $9 → $15, per Trustpilot reviews). Existing subscribers were charged the new rate without opt-out per reviews. This is a live pain point.
- **Beacons** changed pricing structure 2023 → 2024 (modular Pro bundles). No recent global hike documented.
- **Stan.Store** discontinued the Funnels feature ~Feb 2025 with no price reduction, which drew complaints.
- Others: no notable hikes documented.

**Implication for tadaify:** "Price-lock guarantee" (price you signed up at, forever) is a credible hook against Linktree given the November 2025 hike. Document this in onboarding copy.
