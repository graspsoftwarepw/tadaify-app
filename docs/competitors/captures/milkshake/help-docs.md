## Functional area coverage

### Onboarding
- **Mobile-only** — users download app from App Store / Play Store,
  sign up in-app
- **"Shake it up"** randomiser for design variations
- Template-first — pick a card-style starting point

### Editor
- **Card-based** — pages are collections of swipeable cards
- Each card has its own layout: text, image, video, links, about,
  testimonial, shop, etc.
- Customise: colors, fonts, logos, layout per card
- "No customisation beyond colors and fonts" — reviewer critique
  [source: creator-hero] — Milkshake's customisation is SHALLOW but
  the card-structure makes it look rich

### Monetisation
- **Shop card** displays products but does NOT process payments
  [source: creator-hero]
- Creators link out to Shopify / Etsy / Stripe
- No Milkshake-native checkout

### Analytics
- 30-day analytics on Free
- **365-day analytics** on Pro+
- Daily / monthly breakdowns of link clicks + card views [direct fetch]
- GA4 integration on Pro+

### Integrations
- Mailing list (Mailchimp / Klaviyo / similar) on Pro
- Meta Pixel on Pro
- Google Analytics on Pro+
- Instagram direct integration (link sync)
- [unclear — Shopify native or link-out]

### Admin / Team
- **Up to 5 websites per account** (account switching)
- No multi-user seats — single account, multiple sites
- Agencies use multiple accounts

### API / Developer
- [unclear — no public API surfaced]

### Migrations / Import
- [unclear — no first-party importer]

### GDPR / Privacy
- Australian company [unclear — verify founder / HQ]
- App-store-hosted subscriptions give Apple/Google data-handling
  layer

## Documented limits

- Websites per account: **5**
- Cards per website: **unlimited** on all tiers
- Links per card: unlimited
- Analytics retention: 30 days (Free) / 365 days (Pro+)

## Hidden features

- **"Shake it up"** — design randomiser; refresh to get a new
  colour+font+layout combo
- **Up to 5 websites** — creators manage multiple brand-specific
  pages from one app
- **Marketing campaign builder** (Pro) — schedule themed card
  rotations
- **Meta pixel + GA4** on higher tiers
