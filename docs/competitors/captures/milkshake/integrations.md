# Milkshake — integrations catalogue

## Email / ESP
- **Mailchimp / Klaviyo / similar** mailing list integration on Pro
  (details [unclear] — Milkshake markets "mailing list integration"
  without enumerating all supported ESPs)
- Contact forms on Pro

## Analytics / Pixels
- **Meta Pixel** (Facebook) — Pro
- **Google Analytics (GA4)** — Pro+
- Built-in Milkshake analytics (all tiers, differentiated by retention)

## Commerce
- **No Milkshake-native checkout**
- Shop card displays products; checkout happens at external provider
  (Shopify / Etsy / Gumroad / Stripe Link)
- Link-out is the dominant commerce pattern

## Social platforms
- **Direct Instagram integration** — link sync, IG feed pull
  [unclear — depth]
- TikTok
- YouTube embeds
- Twitter/X embed

## Media embeds
- YouTube video cards
- Spotify / SoundCloud audio cards
- [unclear — specific embed catalogue]

## Multi-site
- Up to **5 websites per account**
- Account switching built into the app

## Custom domain
- Pro+ only
- DNS CNAME setup

## Gaps vs competitors

- **No Stripe / PayPal native** — shop card is display-only
- **No Shopify storefront embed** (link-out only)
- **No Instagram DM automation**
- **No public API**
- **No web-based editor** — desktop users cannot edit Milkshake sites
  (this is a feature, not a bug, per Milkshake's positioning, but
  it's a gap for creators who prefer desktop)

## Integration strategy

Milkshake's integration surface is **deliberately narrow**. The
product's brand promise is "build on your phone in minutes" —
adding deep integrations would pull creators off-phone and into
external tool setups, diluting the promise. This is a deliberate
constraint.
