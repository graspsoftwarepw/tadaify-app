# Milkshake — pricing matrix

## Tier matrix (April 2026)

| Tier | Monthly | Annual | Key features |
|------|---------|--------|-------------|
| Free | $0 | $0 | Unlimited cards, links, 30-day analytics, Milkshake branding |
| Lite | $2.99/mo | [unclear] | Remove Milkshake branding |
| Pro | $6.99/mo | $59.99/yr | Mailing list, contact forms, SEO tools, Meta pixel, 365-day analytics, marketing campaign builder |
| Pro+ | $9.99/mo (~$29.99 quarterly) | $99.99/yr | Custom domain, Google Analytics, all Pro features |

Sources:
- [milkshake.app/pricing](https://milkshake.app/pricing/)
- [help.milkshake.app — subscription comparisons](https://help.milkshake.app/en/articles/9478349-milkshake-subscription-comparisons)
- [Creator Hero — Milkshake pricing](https://www.creator-hero.com/blog/milkshake-pricing-and-review)

Annual is a significant discount at Pro ($59.99/yr vs $83.88/yr
monthly ≈ 28% off) and Pro+ ($99.99/yr vs $119.88 ≈ 17% off).

## Free plan

From direct fetch:
- **Unlimited cards** (each card is a swipeable page section)
- **Unlimited links**
- **30-day analytics** (30-day retention)
- Core customisation (colors, fonts) on free tier
- Milkshake-branded card appended to the page
- Site creation + hosting free forever

Source: direct fetch of milkshake.app + [limey.io/blog/linktree-vs-milkshake]

**Milkshake's free tier is among the most feature-rich in the
category** — creators can build a fully-functional mobile website
without paying.

## Lite ($2.99/mo)

- Removes Milkshake branding
- Otherwise same as free
- Cheapest "branding removal" paid tier in the entire research set

## Pro ($6.99/mo or $59.99/yr)

- Contact forms (email collection)
- Mailing list integration (Mailchimp / Klaviyo / similar)
- SEO tools (meta tags, page titles)
- **Meta (Facebook) Pixel tracking**
- **365-day analytics**
- Marketing campaign builder
- Priority support

## Pro+ ($9.99/mo or $99.99/yr)

- Custom domain connection
- **Google Analytics** integration
- All Pro features
- Card hiding / unhiding [per help-centre subscription-comparisons
  article]

## Transaction fees

- **No Milkshake platform fee** — Milkshake is not a commerce platform
- Shop card is a feature for displaying products, but "you can't
  process orders and payments on the platform" [source:
  https://www.creator-hero.com/blog/milkshake-vs-linktree-features-pricing-which-one-is-best-for-you]
- Creators link to external commerce (Shopify, Etsy, Gumroad, Stripe
  Links)

## Anchor pattern

- **Lite at $2.99 is the cheapest branding-removal tier** across the
  entire research set
- **Pro at $6.99** is cheaper than Linktree Starter ($8) while
  including Meta Pixel + 365-day analytics + marketing campaigns
- **Pro+ at $9.99** matches Beacons Creator but bundles custom domain
  + GA4 — strong value

## Comparison lens

| | Milkshake Pro $6.99 | Linktree Starter $8 | Campsite Pro $7 |
|---|--------------------|---------------------|-----------------|
| Custom domain | ✗ (Pro+ only) | ✗ | ✓ |
| Meta Pixel | ✓ | [unclear] | ✓ |
| 365-day analytics | ✓ | [shorter] | ✓ |
| Email capture | ✓ | ✓ | ✓ |
| Card-based design | ✓ (unique) | ✗ | ✗ |
| Mobile-only | ✓ | web+mobile | web+mobile |

Milkshake Pro is competitive with Linktree Starter / Campsite Pro on
feature surface, with the card-based design as the differentiator.
