## Article groupings by functional area

### Onboarding
- New-user signup flow articles covered under Account support [unclear —
  specific article titles]
- Theme selection: "Choose a theme for your Linktree" (article 5434137)
  [source: https://help.linktr.ee/en/articles/5434137-choose-a-theme-for-your-linktree]

### Editor
- 95-strong "How-to's" category is the bulk of the help corpus
- Link types (classic link, header, commerce, digital product, tip jar,
  paid content, video embed, social icons) — each with its own article
  [unclear — enumerate from live crawl]
- Theme builder articles covering buttons, fonts, background

### Monetisation / Payments ("Earn with Linktree")
- 21 articles covering: Digital Products, Tips, Shops (affiliate), Paid
  access / Memberships, Payout setup (Stripe / PayPal connection),
  Tax forms, Seller fee structure
- Articles document that checkout is handled inline through Linktree's
  Stripe / PayPal connection — creators link their own processor
  [source: multiple review syntheses]

### Analytics ("Insights")
- "Understanding your Insights" — page-level overview article (5434178)
- "Individual Link Insights" — per-link performance (5434172)
- "Google Analytics Integration" — connect a GA property (5434201)
- Metrics documented: clicks, CTR, revenue, conversions, device, location,
  referral source, date-range up to 365 days on paid tiers
  [source: Linktree monetization suite TechCrunch]

### Integrations
- Each marketplace integration has a paired help article describing setup
- Mailchimp, Klaviyo, Google Sheets (email forwarding), Spotify embed,
  YouTube embed, TikTok, Shopify storefront, Calendly, Zapier
  [source: multiple review sites]

### Admin / Team
- "How to add an admin user to manage your Linktree" (article 8609951)
  [source: https://linktr.ee/help/en/articles/8609951-how-to-add-an-admin-user-to-manage-your-linktree]
- Admin user = delegated editor role; avoids sharing primary login
- Seat counts per tier [unclear — needs live audit; historically
  admin users were Pro+ only]

### API / Developer
- No dedicated help-centre category for API
- Dev content lives at `linktr.ee/api` and `linktr.ee/marketplace/developer`
- Help centre does NOT document public-API endpoints (developer-only
  surface outside the user help)

### Migrations / Import / Export
- No prominent "import from Beacons / Bio.link / Carrd" article surfaced
  in search [unclear — Linktree may not offer first-party migration
  tooling]. Competitors leverage this gap (Beacons/Pallyy advertise

- Export of user data covered under GDPR policy articles [unclear — verify]

### GDPR / Privacy

- Includes DSAR / data-export path (GDPR Art. 15 + 20)

### Troubleshooting
- "I can't log in to my account. What should I do?" is called out as a

- Browser cache / cookie issues articles
- Content moderation / account locked articles (connected to reported

## Documented limits / quotas / rate limits

- **Character limits**: Bio description limited to 80 characters on free
  plan [source: Capterra reviews] [unclear — whether this is a hard
  product limit or a free-tier restriction]
- **Link count**: "unlimited" on all tiers [source: talkspresso]
- **Analytics retention**: "last 365 days" is the paid-tier window for
  per-link analytics [source: TechCrunch 2025 monetization launch post]
- **File upload size** for digital products [unclear — not publicly
  documented]
- **Webhook rate limits** [unclear — no public docs]
- **API rate limits** [unclear — no public docs]

## Hidden features surfaced only through docs

- **Instagram auto-post-to-link** automation (Starter+) — auto-creates a
  link whenever a new Instagram post goes live [source: talkspresso]
- **Scheduled links** — time-gated links that go live / expire on a
  schedule [source: thesocialcat]
- **Sensitive-content gating** for age-restricted content
  [unclear — present historically]
- **Custom 404 for claimed usernames** [unclear]
- **Claim-your-username URL reservation flow** (for brands that want to
  park a username) [unclear]
- **Referral codes / creator affiliate** — the Rewards programme surface
  [source: biotree]

## Known issues / complaints referenced in docs or workaround-heavy articles

- Content moderation takedown process — multiple help articles explaining
  the appeal flow imply the takedown volume is non-trivial [source:
  networksolutions.com/blog/linktree-alternatives]
- No first-party custom-domain article — because the feature does not
  exist on any tier [source: daily-ads review 2026]
