## Functional area coverage

### Onboarding
- Quick setup promise
- Template-first: 10+ customizable templates on free
- Fast-load page optimisation is a product-level commitment (hero
  "blazing fast")

### Editor
- Link types: classic link, title, divider, social account, group
  (for nested links)
- Block types for engagement: opt-in forms, carousels, feeds
- Media embeds on Pro (video, audio)
- Auto-Pilot: scheduling, automation (time-gated links)
- Bulk editing on Pro+

### Monetisation
- Integrated commerce flows via embedded blocks
- No native checkout surface
- Support for affiliate tracking (UTM parameters)

### Analytics

  device) [source: reviewer syntheses]

- Tracking pixels (GA / Meta / TikTok)
