# Campsite.bio — templates & themes

## Gallery size

- **10+ templates** on free tier [source:
  https://limey.io/blog/campsite-vs-linktree]
- More templates on Pro [unclear — exact count]

Compared to category:
- Linktree: ~41 (reviewer-cited)
- Carrd: 264
- Taplink: 100+ (templates) + 400+ themes
- Lnk.Bio: 368+
- Campsite: 10+ free, more paid

Campsite's template library is **smaller than competitors in raw
count** but more capable in free-tier customisation.

## Customisation depth (free tier — unusual generosity)

Per [limey.io/blog/campsite-vs-linktree], free tier allows:
- Upload own images for backgrounds
- Adjust fonts
- Customise text spacing
- Customise social icon placement
- 10+ templates to start

This is materially more flexibility than Linktree / Beacons free
tiers.
