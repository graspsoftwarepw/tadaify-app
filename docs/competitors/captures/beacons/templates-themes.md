# Beacons.ai — templates & themes scan

## Gallery / entry points

- "Design your Page 🎨" KB article (primary customisation guide):
  [help.beacons.ai/en/articles/4697153](https://help.beacons.ai/en/articles/4697153)
- "Customizing your page" blog tutorial:
  [beacons.ai/i/blog/customizing-your-page](https://beacons.ai/i/blog/customizing-your-page)
- No public template gallery at a well-indexed URL slug like
  `/templates` or `/themes` — Beacons' template surface lives **inside
  the editor** rather than as a SEO landing
  [source: absence in search results]

This is a meaningful strategic difference vs Linktree (which runs
`/s/templates` as a public SEO surface) and Carrd (which has a public
cloneable-template community). Beacons appears to deprioritise
template SEO in favour of in-app template discovery.

## Theme / customisation capabilities (documented)

From reviewer synthesis + direct fetch of `/i/plans`:

- **1024+ fonts** to choose from [source:
  https://freerdps.com/blog/beacons-ai-review/]
- Unlimited customisation of colors, fonts, styles, layouts —
  promised on ALL tiers including Free [source: direct fetch
  `beacons.ai/i/plans`]
- Per-block customisation: **shape, shadow, outline, transparency**
  for each block [source: freerdps, digitalsoftwarelabs]
- **Backgrounds**: colour, gradient, uploaded image, uploaded
  **video** [source: freerdps]
- **Mobile-optimised** drag-and-drop template builder [source:
  multiple reviews]
- **Rainbow / custom theme** option — a "click this for more
  customisation" control that exposes deep colour / gradient pickers
  [source: freerdps]

## Customisation depth ceiling

Per [digitalsoftwarelabs.com/ai-reviews/beacons-ai/]:

> "It's important to note that CSS/HTML access is not available, so
> you're locked into the platform's design structure. This means you
> cannot add custom CSS code directly to your Beacons page."

- **No custom CSS / HTML** — confirmed
- **No self-hosted @font-face** — but the in-product font catalogue
  (1024+) is large enough that it covers most brand needs
- **No per-link custom classes** — block styling is page-level only
  [unclear — confirm in Design your Page KB]
- **Multiple pages supported** (up to 10-per-site on Creator Max, 3
  on Creator Plus, 1 on lower tiers) — richer than Linktree's strict
  single-page model [source: direct fetch `/i/plans`]

- Link-in-bio (single-page)
- Website (multi-page)
- Media kit (brand-deals facing)
- Store / product showcase
- Event / booking page

## Notable / premium templates

- **Rainbow colour theme** — hero-level animated palette option
  [source: freerdps]
- **Video-background templates** — use of uploaded video as hero
  [source: freerdps — unique among the major competitors at free-tier
  level]
- Seasonal / event templates [unclear — not surfaced in search]

What makes a template "premium-feeling" on Beacons:
- Motion (video background, animated gradients)
- Multi-page layout (vs single-screen scroll)
- Embedded store / media-kit sections
- Integration Block embeds (Ko-fi panel, Split commerce grid) inline

## Mobile preview

The editor includes a device-preview switch — creators can see mobile
rendering live [source: multiple reviews describing the drag-and-drop
experience with live preview]. Exact preview device breakpoints

## Theming depth comparison (snapshot)

| Capability | Beacons | Linktree | Carrd |
|-----------|---------|----------|-------|
| Custom CSS | ✗ | ✗ | ✓ (Pro) |
| Font options | 1024+ | ~20-50 | Webfont picker |
| Video background | ✓ | ✗ | ✗ |
| Multi-page | ✓ (up to 10) | ✗ | ✓ (Pro) |
| Animated themes | ✓ | ✓ (Pro) | ✗ |
| Public template gallery SEO | ✗ | ✓ | ✓ |
| In-editor template picker | ✓ | ✓ | ✓ |

Beacons wins on **editor depth**, loses on **template discovery /
SEO surface**.

## Strategic observation (neutral)

Beacons' lack of a public `/templates` gallery leaves SEO airspace
that a tadaify-style entrant could occupy — ranking on creator-niche
template searches ("musician link in bio template", "small business
link in bio template") that Beacons does NOT actively rank for.
