# Later Linkin.bio — help docs scan

## Help-centre structure

Later has a comprehensive help centre at `help.later.com` [unclear —
exact slug]. Linkin.bio documentation is a sub-section inside Later's
broader help corpus (alongside scheduling, analytics, team features).

## Group by functional area

### Onboarding (Linkin.bio specifically)
- Linkin.bio setup: connect IG/TikTok/Snap, customise design, publish
  page
- Grid-match feature: Linkin.bio page mirrors IG feed; each IG post
  has its own clickable link (Starter) or up to 5 links (Growth+)

### Editor
- Custom design — buttons, backgrounds, text colors, themes
- Featured banner block for top content
- Pre-designed themes [limited count — reviewer critique]

### Monetisation / Commerce
- **Shopify product tagging** — creators connect Shopify store, tag
  products in IG posts, tagged products render in Linkin.bio with
  checkout flow via Shopify
- Affiliate links: custom URL links with tracking
- No Later-native store / digital products

### Analytics
- **Post-level link clicks** — unique to Later: every IG post has its
  own Linkin.bio analytics
- Pageviews
- Click-through rates
- Sales per post (via Shopify integration)
- Follower trends (Later's scheduler-level analytics)

### Integrations
- **Instagram, Facebook, TikTok, Pinterest, LinkedIn, Snap** —
  native on Later's scheduler; Linkin.bio uses IG/TikTok/Snap feeds
- **Shopify** — deep integration, product tagging
- **Canva** — design creation inside Later
- **Dropbox** — media import
- **Mailchimp** — email capture on Linkin.bio page (direct fetch
  confirmed)
- **Google Analytics** [unclear — verify]
- Later has no "Integration Block" architecture — integrations are
  feature-level, not partner-extensible

### Admin / Team
- Team collaboration with approval workflows (higher tiers)
- Multi-seat model built into the scheduler

### API / Developer
- [unclear — Later may have enterprise API for brands/agencies, but
  no public dev portal]

### Migrations / Import
- Content library imports from social platforms (IG / TikTok feeds
  auto-pull)
- No "migrate from Linktree" [unclear]

### GDPR / Privacy
- Later is GDPR-compliant (they're headquartered in Vancouver,
  Canada, with European operations)
- Standard DSAR / export flows [unclear — verify]

## Documented limits

- Link-per-post: 1 (free/Starter) / 5 (Growth+)
- Social sets: 1 (Starter) up to 15+ (Agency)
- Scheduled posts: unlimited on most tiers [unclear — verify caps]
- Analytics retention: Later tier-dependent [unclear]

## Hidden features

- **Grid-match rendering** — Linkin.bio page visually mirrors IG grid;
  creators' audiences experience continuity
- **Per-post analytics** — you know which post drove which click;
  most competitors give aggregate link-level analytics only
- **Shopify product tagging** — deeper-than-most commerce path

## Support channel

- Help centre
- Email support
- Priority support on higher tiers
- Later has an established content-marketing arm (blog posts, creator
  webinars) that doubles as help content
