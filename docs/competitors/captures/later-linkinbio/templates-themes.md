# Later Linkin.bio — templates & themes

## Theming surface

Linkin.bio's defining template is the **IG grid mirror** — the page
renders a grid that matches the creator's Instagram feed. Each grid
tile is clickable and can route to a different URL.

This is arguably the most distinctive UI pattern in the category —
no competitor (Linktree, Beacons, Stan, Carrd, Hopp, Taplink,
Milkshake, Lnk.bio, Campsite) uses "the IG grid IS the bio page" as
its primary visual.

## Featured banner

- Top-of-page banner that highlights a time-sensitive promo or hero
  content
- Directs traffic to a priority link

## Grid mirror behaviour

- Linkin.bio page auto-populates with the creator's IG posts (or
  TikTok/Snap posts)
- Each post tile can be assigned a different destination URL
- On **Growth+**, each post can have **up to 5 links** (e.g. the
  post features 3 products; each product links to its own page)

## Design ceiling

- No custom CSS
- No custom fonts beyond Later's theme library
- No multi-page
- No animated backgrounds
- Structure is hard-coded to grid / banner / featured-block layout

## Mobile preview

- Baked into Later's editor (Later is mobile-first across all products)
- Creators can see mobile + desktop preview side-by-side
