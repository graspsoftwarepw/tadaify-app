# Later Link in Bio (Linkin.bio) — pricing

## Structure

Linkin.bio is **bundled with Later subscriptions** rather than
standalone-priced. Later's tiers determine what Linkin.bio
capabilities are unlocked.

- **Free Linkin.bio** — basic page, 1 linked URL per post
- **Starter plan** — $25/mo (or $20.83 on annual)
- **Growth plan** — [unclear — ~$45/mo; adds up to 5 links per post]
- **Advanced plan** — [unclear — mid-tier]
- **Agency plan** — $200+/mo (for agencies with 15+ social sets)

Source: [cutedyno — Later pricing plans 2026](https://cutedyno.com/pricing/later),
[socialrails — Later pricing 2026](https://socialrails.com/blog/later-pricing),
[fahimai — Later $25/mo](https://www.fahimai.com/later-social-media-scheduler).

Annual billing saves 20-33% depending on tier.

## Free Linkin.bio tier

- Basic link-in-bio landing page
- One linked URL per post
- Limited analytics
- Later / Linkin.bio branding
- Intent: lead-gen for Later's paid scheduler

## Starter ($25/mo)

- 1 social set
- Linkin.bio basic features
- Scheduler for 1 IG / 1 FB / 1 TW / 1 Pinterest / 1 TikTok / 1 LinkedIn
- Visual content calendar
- Instagram grid preview
- Basic analytics

## Growth plan (~$45/mo)

- **Up to 5 links per Instagram post** — unique Linkin.bio capability
- Shopify product tagging
- Deeper analytics
- Multiple social sets
- Team features (small)

## Advanced / Agency

- 15+ social sets
- White-label options [unclear]
- SSO / enterprise features [unclear]
- Workflow approvals

## Transaction fees

- **No platform fees** on sales — because Linkin.bio is a link page,
  not a checkout surface. Commerce flows through Shopify (tagged
  products route to Shopify checkout)
- Shopify's standard fees apply

## Pricing position

Later's $25 entry is **the highest in this research set for a
link-in-bio-adjacent product**. But that's because Later is primarily
a social-media scheduler; Linkin.bio is a feature bundled in a
$25-$200/mo product.

Comparison:

| | Later Starter $25 | Linktree Pro $15 | Beacons Creator Plus $30 |
|---|------------------|------------------|--------------------------|
| Scheduling | ✓ (full) | ✗ | ✗ |
| Link in bio | ✓ | ✓ | ✓ |
| 5 links per post | ✗ (Growth+) | ✗ | ✗ |
| Shopify product tagging | ✓ | [unclear] | ✗ |
| Analytics | deep | medium | deep |
| Value for JUST link-in-bio | poor | good | good |

Later is **only worth it if you need the scheduler**. As a
link-in-bio-only tool, it's expensive.

## Free plan

- **Linkin.bio free exists** as a standalone offering — creators can
  use it without a paid Later sub
- Restricted to single link per post
- Limited analytics
- Later branding
