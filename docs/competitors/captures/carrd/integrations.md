# Carrd — integrations catalogue

## Integration philosophy

Carrd is **embed-first, not connector-first**. Its integration model
is: if a service gives you an iframe or a `<script>` tag, Carrd will
host it. Carrd doesn't maintain a partner directory — it's intentionally
backend-agnostic.

## Forms & email
- **Mailchimp** — native form integration
- **ConvertKit (Kit)** — native form integration
- **Zapier** — native form-submit action
- **Google Sheets** — native form-submit destination
- **Direct email** — forms email to creator inbox
- **Buttondown / other niche ESPs** — via Zapier or custom webhook

in reviewer comparisons.

## Third-party widgets / embeds

Because Carrd supports unrestricted iframe / widget embeds, creators
effectively have access to:
- **Gumroad** buy-buttons
- **Stripe Checkout / Stripe Payment Links**
- **Shopify Buy Button**
- **Calendly / Cal.com / SavvyCal / Zcal** booking embeds
- **Substack subscribe embed**
- **Beehiiv embed**
- **Notion public-page embed**
- **YouTube / Vimeo / Loom** video
- **Spotify / SoundCloud / Apple Music** audio
- **Twitter / Instagram / TikTok** post embeds
- **Tally / Typeform / Jotform** form embeds

Carrd users report mixing-and-matching 3-5 widgets per page as normal
pattern.

## Analytics
- **Google Analytics (GA4)** native
- **Meta (Facebook) Pixel** via custom-code / Pro Plus
- **Plausible / Fathom / Simple Analytics** via custom-code / Pro Plus
- Carrd-native analytics on Pro Standard+

## Ad networks
- **No native ad-monetisation surface** (contrast with Hopp's Pre-rolls
  or Beacons' affiliate catalogue)
- Creators embed their own affiliate links as ordinary links

## Social platforms
- Standard link-out; no auto-feed / auto-pull integrations
- Embeds of specific posts via iframe

## Commerce
- **No native checkout** — everything is embed
- Creators most commonly use Gumroad + Stripe Links for simple
  commerce on Carrd pages

## Domain / DNS
- **Let's Encrypt SSL** automatic on Pro Standard+
- DNS management at the registrar (Carrd doesn't sell domains)
- Multi-domain per site [unclear — verify]

## Partnerships
- Carrd does not have obvious co-marketing partnerships
- Partnership model: none — Carrd is indie and intentional about it

## Gaps vs competitors

- **No native commerce platform** — creators relying on Carrd for
  sales must pair with Gumroad / Stripe Links
- **No analytics depth** comparable to Beacons' post-performance /
  income management
- **No email-sending** (Carrd collects emails, never sends them)
- **No media kit / brand-deals surface**
- **No Instagram DM automation**

## Why the gaps don't matter to Carrd's users

Carrd's ICP is builders who would rather assemble best-of-breed tools
than adopt an all-in-one. Gap is feature, not bug.
