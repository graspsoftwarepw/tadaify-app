# Carrd — templates & themes

## Gallery

- Carrd opens with a template picker ("Choose a Starting Point") as
  the primary CTA [source: direct fetch carrd.co/]
- **264 templates** — the largest library in the category
  [source: https://www.mobilocard.com/post/linktree-vs-carrd]
- Free tier includes basic templates
- Premium + custom templates gated to Pro Lite+ ($9/year)

## Template categories

[unclear — exact taxonomy in gallery; Carrd uses descriptive names not
niche categories]. Reviewer summaries indicate sub-genres:
- Personal portfolios
- Landing pages
- Event pages (weddings, conferences)
- Product launches
- Coming-soon pages
- Link-in-bio style single-column
- Photography portfolio
- Small business one-pager
- Resume / CV sites
- Newsletter landing pages
- Non-profit / fundraiser pages

## Customisation depth

- Block-based editor with container nesting
- **Advanced settings tab** on Pro Plus unlocks custom code
- **Custom CSS** injection (Pro Plus)
- **Custom JS** injection (Pro Plus)
- **HTML embed blocks** any tier
- Multi-section pages (Carrd pages ARE single-page but can have many
  scroll sections — contrast with Linktree's truly-linear link list)
- Responsive breakpoints (auto-handled, not creator-specified)
- Animations / transitions on elements [unclear — verify Pro features]

## Custom code ceiling

**Carrd is the ONLY tool in this research set that offers custom
CSS + JS on any tier.** This is the fundamental reason designers
pick Carrd over Linktree / Beacons / Hopp.

## Template marketplace (third-party)

Because Carrd has an active user base of designers, a third-party
template market has emerged:
- **Gumroad** — many creators sell Carrd templates ($10-50 per
  template)
- **Envato / ThemeForest** — Carrd-compatible templates
- **Dribbble** — Carrd design inspiration showcases

## Mobile preview

- Carrd editor has explicit mobile-preview mode
- Responsive by default; creators can set element breakpoints

## Dark mode

- Themeable per-template (most templates support a dark variant)
- No automatic `prefers-color-scheme` toggle by default [unclear —
  Pro Plus custom CSS can enable]

## What makes a Carrd template "premium"

- Multi-section long-scroll layout (not just link list)
- Animation / parallax effects
- Pre-configured forms with validations
- Colour-scheme systems rather than flat palettes
- Typography pairings chosen for aesthetic consistency
