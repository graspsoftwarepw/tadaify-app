# Carrd — help / documentation scan

## Documentation root

- Primary docs: [carrd.co/docs](https://carrd.co/docs)
- Pro-specific plans page: [carrd.co/docs/pro/plans](https://carrd.co/docs/pro/plans)
- Go-Pro landing: [carrd.co/pro](https://carrd.co/pro)

Carrd's documentation is **developer-oriented** and relatively terse
compared to category norms (Linktree's 181-article help centre, Beacons'
180+). Carrd users are expected to be technically comfortable.

## Group by functional area

### Onboarding
- "Choose a Starting Point" = template picker at signup
- 264+ templates — largest template gallery in the research set
  [source: Mobilo Linktree-vs-Carrd comparison]
- Empty-page option for blank-start builders

### Editor
- Block-based editor (not drag-drop-pixel like Webflow; more like
  constrained blocks)
- Elements: text, image, container, gallery, slideshow, form, embed,
  widget, divider, spacer
- No Figma-style canvas; Carrd's opinion is "one-page = linear scroll"

### Custom code (Pro Plus)
- JavaScript injection for advanced behaviour
- CSS injection for styling beyond Carrd's controls
- HTML embed blocks for arbitrary content
- **Download source**: on Pro Plus creators get unminified HTML/CSS/JS
  of their site — a feature no competitor in this set matches

### Forms + email integration (Pro Standard+)
- Native form builder
- Email integrations: Mailchimp, ConvertKit, Zapier, Google Sheets,
  direct email
- Webhooks on form submit [unclear — verify on Pro Plus]
- Conditional fields [unclear]

- Site analytics baked in
- Google Analytics (GA4) integration

## Documented limits / quotas

- Sites per plan: 3 / 3 / 10 / 25 (Free / Pro Lite / Standard / Plus)

## Hidden features

- **Source code download** (Pro Plus) — rare in SaaS website builders
- **No vendor lock-in** — creators can migrate sites off Carrd by
  downloading sources
- **264+ templates** — category-leading count
- **Embeds unrestricted** — any iframe / widget works (commerce,
  calendar, forms, etc.)
