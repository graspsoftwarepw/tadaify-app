# Bio.link (Hopp by Wix) — help centre scan

## Help-centre structure

Hopp has **two parallel help surfaces** because of its multi-parent
ownership:
1. **Wix help centre** — primary for feature guidance, since Wix owns
   Hopp: [support.wix.com/en/article/hopp-by-wix-*](https://support.wix.com/en/article/hopp-by-wix-adding-and-managing-actions-for-link-in-bio)
2. **Name.com knowledge base** — because Name.com distributes Hopp
   and sells the .bio/.link domains:
   [name.com/support/articles/23862806330387-hopp-by-wix](https://www.name.com/support/articles/23862806330387-hopp-by-wix)

This is unusual in the category — most competitors have a single
help-centre root.

## Articles of strategic interest

- "Hopp by Wix: Adding and managing Actions for link in bio" — main
  how-to for adding link types and blocks
- Hopp Stripe payment settings (via Wix support) — payment setup

## Group by functional area

### Onboarding
- Wix-native onboarding flow — Hopp is positioned as an add-on /
  companion to Wix, so creators with Wix accounts get a smoother
  flow
- Name.com users who buy a .bio or .link domain get a Hopp-onboarding
  funnel on checkout completion

### Editor
- Intuitive editor with professional templates
- Widgets as the unit of composition — products, forms, email capture
- Customisable search engine as a unique editor component
- Page layout, design, navigation, settings all customisable
  [source: search result from wix.com/blog/what-is-hopp]

## Hidden features

- **Customizable search engine over your own links** — genuinely
  unique; no other competitor has this
- **Pre-rolls** — Hopp's native ad format; creators monetise
  impressions directly (not via transaction fees)
- **Wix-integration depth** — Hopp users who also use Wix websites
  get unified Wix store sync, unified analytics [unclear —
  confirm depth]
- **Branded short-link shortener** — bundled utility that competitors
  sell as a separate product (Bitly / Rebrandly)
