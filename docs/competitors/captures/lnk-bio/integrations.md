# Lnk.Bio — integrations catalogue

## Email / ESP
- **Mailchimp** — native email-capture integration
- Other ESPs [unclear — possibly via Zapier, though Zapier
  connectivity not prominently documented]

## Analytics
- Built-in basic analytics on free
- Advanced analytics on paid (Unique / Ultra)
- Google Analytics [unclear — verify]

## Social platforms
- Instagram / TikTok / YouTube / Spotify / Facebook / X — standard
  link-out
- [unclear — auto-feed-pull like Linktree's IG sync]

## Commerce / payments
- Embeddable widgets for Gumroad / Stripe / similar link-outs
- **No native Lnk.Bio checkout** — Lnk.Bio is a link-page, not a
  commerce platform

## Custom domain
- Supported on Unique / Ultra
- DNS CNAME setup

## Video / media
- Video wallpaper support (unusual)
- YouTube / Vimeo embed via link block

## Gaps vs competitors

- No Stripe-native payments
- No Shopify storefront embed
- No Instagram DM automation
- No native email-sending (Mailchimp capture only)
- No affiliate / brand-deals catalogue
- No media kit

## Integration strategy

Lnk.Bio's thin integration surface reflects its positioning: cheapest
tool, not broadest tool. Creators adopt Lnk.Bio for the lifetime deal
+ design flexibility, accepting the lighter integration surface.
